# Problem

Write a function rotate(ar[], d, n) that rotates arr[] of size n by d elements.

ex: we have array[1,2,3,4,5,6]

--> rotate(array, 2 ,6) 
----> array[3,4,5,6,1,2]


# Idea

1. The worst case 

--> Create a new array same size 

--> set and copy to the final

2. Swap method

--> get the first element, swap to the end of array
